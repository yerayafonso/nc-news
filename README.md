# Northcoders News App

This is a mock News application that allows users to browse articles, view individual articles, vote on articles, post comments and delete comments. Posting and deleting can only be done if ther is a user logged in.

This is a mock News web application that is built with:

- React
- JavaScript
- HTML
- CSS
- Node.js
- Vite

# Live Sites

Link to deployed version: https://nc-news-ya.netlify.app/

Backend API - https://nc-news-app-5h3i.onrender.com/api/

## Features

- News article retreival
- View individual article
- Article Voting
- Commenting System
- Comment Voting
- View articles by topic
- Sorting

## Installation

**Minimum Node version required:** 25.2.1

To run this project locally:

1. Clone the repo

```
git clone https://github.com/yerayafonso/nc-news.git
```

2. Navigate into the directory:

```
cd nc_news
```

3. Install dependencies

```
npm install
```

4. Start the server

```
npm run dev
```
